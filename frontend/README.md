# Super Copias - Frontend (esqueleto)

Estructura base para el frontend con Angular. Este esqueleto contiene módulos y componentes iniciales para:
- Login
- Admin
- Clientes
- Empleados

Pasos para iniciar localmente:
1. cd frontend
2. npm install
3. npm start

Notas:
- Este repositorio asume que el backend corre en el mismo host y puerto (ej: proxied) o que configures `proxy.conf.json` para redirigir `/api` al backend.
- Completa los componentes y estilos usando el proyecto de referencia `referencias/sb-admin-pro-angular` para la UI.
